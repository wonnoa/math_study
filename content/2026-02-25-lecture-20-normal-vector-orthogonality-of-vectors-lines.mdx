normal vector : graph에 대하여 90도인 벡터가 왜 생기는지를 알려면, 90도가 무엇인지 알아야한다.

알고자하는 것 :
    90도가 왜 '관계없음'인가.
    각도를 시각적인 요인을 배제하고 '수'로만 설명하려면 ?



- 무엇인지 이해하기 위해서, 그냥 각각 90도인 선분을 만들어보기로 한다.
    
    인간의 시각적인 요인을 배제하고, 90도란 무엇인가.
    앞을보지 못하는, 한번도 무엇인가를 본적없는 사람에게 어떻게 90도를 설명할 것인가?
    지극히 수의 논리적 연결성 관점에서, 90도는 수와 수의 어떠한 관계로 만들어지는 것 인가?

    90도를 하나의 기준 체계인것으로 받아들여보기로 한다.
    90도를 unlearn해야하겠다는 생각도 해본다.

수의 관계 :

    이 때, 45도는 특정 수와 다른 수의 증가비율이 1:1관계인 것을 의미한다.
    다만 45도 각도를 만들어내는 기준점이 필요한데, 이것은 x축일 것 이다.
    x축의 equation은 1,0이다. 이를 상수배하면 2,0이다.

벡터와 벡터의 90도 관계 :

    단순 x와 y의 증가비율의 관계라고만 하기에는 벡터는 하나의 함수이다.
    그래서, 벡터들이 90도 관계가 되려면 equation간의 관계가 90도가 되어야하는 일이 발생한다.

Graph를 본적도, 볼수도 없는 나. 90도와 graph를 잊은 후, 나중에 다시 90도라는 unit 체계를 적용하기로 한다 :

    두 개의 수 뭉치가 주어졌다.
    
    set 1 : [<1,0>, <2,0>, <3,0>]
    set 2 : [<0,1>, <0,1>, <0,3>]

    그래프로 그리려는 instinct를 멈춘다.
    여기서 도출할 수 있는 것은, 두 개의 '규칙 체계'가 존재한다는 것,
    그리고 이 '규칙 체계'에서 수와 수의 증가 관계를 도출해볼 수 있다는 것 이다.

    set 1 : 첫번째 배열에 있는 수가 증가할 때, 다른 배열의 수 가 전혀 증가하지 않는 관계를 가지고 있다.
    set 2 : 두번째 배열에 있는 수가 증가할 때, 다른 배열의 수 가 전혀 증가하지 않는 관계를 가지고 있다.    

    그래 이 두개의 관계는 90도관계이구나.

두 개의 규칙체계에서, 수들의 증가관계는 어떻게 다를까? :

    음.
    예를 들어 아래처럼 수의 규칙이 있다고 치면.
    set 1 : <1,0>, <2,0>
    set 2-1 : <1,1>, <2,2>

    자 이것은 그래프를 못 본사람은 어떻게 그들이 소위 90도인지 아닌지를 알 수 있는가?
    dot product로 증명하려 하지말아라, 그 전에 수의 관계측면에서 이 수들의 규칙들이 도대체 어떤 관계이고, 어떻게 비교되는가가 중요하다.

    다만 전제조건이 하나 깔려야할 것 같은데.
        1. 각 기준 차원은 unit degree로 놓여있다.
    
    
원시인이 되어본다. 나는 각도도 모르고, 90도모른다.



최종적으로 unit vector가 아니어도 90도가 되는 vector들, 혹은 equation들은 어떤 수의 관계를 가지고 있는 것 일까? :




** 부산물 : dot product

    [a * x] / [b * y]
