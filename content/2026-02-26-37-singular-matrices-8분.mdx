# 37 Singular Matrices (8분)

- 정의
    
    ```yaml
    Singular Matrix : Inverse가 없는 Matrix
    
    Non-Singular Matrix : Inverse가 있는 Matrix
    ```
    
- 부가 설명
    
    ```yaml
    Singular Matrix invert하려면 마치 0으로 무엇인가를 나누는 것과 같다.
    
    - 0x = 5일때, x를 찾을 수 있는가?
    
    한마디로, solution을 찾을 수 없다는 것.
    
    블랙홀의 경우, density가 무한하면, volume을 찾을 수 있는가?와 같은 것.
    ```
    
- 머신러닝에서 왜 중요한가
    
    ```yaml
    Matrix 코딩할 때, 완전히 다른 것을 해야할 때가 있다.
    
    Singular = non-invertible 하다.
    ```
    
- Inverse가 없다는 것이 무엇인가 (예시)
    
    ```yaml
    A와 A^-1을 곱해서 I를 찾아야 한다.
    그런데, I를 찾을 수 있는 A^-1이 없다...
    
    unique 한 solution 찾을 수 없음.
    
    예시 : 1111 matrix
    
    A = 
    [1 1]
    [1 1]
    ```
    
    ```yaml
    GPT 설명 :
    
    먼저 연립방정식을 행렬로 표현하는 것부터 시작해보겠습니다:
    AX = B 형태의 방정식에서, 우리는 X를 구하고 싶습니다. 일반적인 대수에서처럼, 양변에 A의 역행렬을 곱하면 해를 구할 수 있을 것 같습니다:
    A⁻¹(AX) = A⁻¹B
    (A⁻¹A)X = A⁻¹B
    IX = A⁻¹B
    X = A⁻¹B
    이것이 역행렬을 이용한 해법입니다. 그런데 여기서 중요한 점이 있습니다. A가 singular matrix라면 이 방법을 사용할 수 없습니다.
    예를 들어 [1 1] 행렬을 생각해보겠습니다:
    [1 1]
    이 행렬로 표현되는 연립방정식은 두 가지 경우가 있습니다:
    
    x + y = 1
    x + y = 1
    이 경우는 같은 방정식이 두 번 나오는 것으로, 해가 무한히 많습니다.
    x + y = 1
    x + y = 2
    이 경우는 모순된 방정식으로, 해가 없습니다.
    
    두 경우 모두 singular matrix와 관련이 있는데, 그 이유는:
    
    singular matrix는 역행렬이 없음
    역행렬이 없다는 것은 X = A⁻¹B 방식으로 해를 구할 수 없다는 의미
    해를 구할 수 없는 이유는 해가 없거나(no solution) 해가 무한히 많기(infinite solutions) 때문
    
    따라서 "행렬이 singular하다"는 것은 단순히 역행렬이 없다는 수학적 사실을 넘어서, 그 시스템이 유일한 해를 가질 수 없다는 더 깊은 의미를 내포하고 있습니다. 이것이 교수님이 역행렬 설명에서 해의 존재 여부를 함께 언급하신 이유입니다.
    ```
