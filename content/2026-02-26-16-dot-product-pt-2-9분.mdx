# 16 Dot Product (pt 2) (9분)

**Linear를 이해해**

```yaml
3 * 2 :
	3 = (1+1+1)
	2 = (1+1)

3이 2개 :
	1+1+1을 한번 더하고, 또 한번 더하고

무엇이 몇 개인개 몇 개인 규칙이 유지되는 것이 선형성.

3곱하기 4가 3곱하기 2 곱하기 2인 이유 :
	
	1. 3이 1개 1개 1개 1개
	2. 3이 1개 1개인것이 1개 1개
```

**Dot Product의 규칙들과 각 규칙의 타당성에 대한 나의 이해**

```yaml
도트 곱의 주요 성질들:

1. 교환 법칙(Commutative Property) :

	x·y = y·x
	이는 두 벡터의 도트 곱 순서를 바꿔도 결과가 같다는 의미입니다.
	재미있는 점은 이 성질이 행렬에는 적용되지 않는다는 것입니다.
	
	*** 당연하다 곱해서 더하는 것 이니까 순서는 상관없지.
	
	matrix에 적용 안되는것도 당연하다.
	곱해서 더해지는 값들이 달라지기때문.
	matrix부터는 다른 차원값끼리 연산하게되어버림.
	
2. 분배 법칙(Distributive Property) :

	x·(y + z) = x·y + x·z
	이는 일반 숫자의 곱셈에서의 분배 법칙과 유사합니다. 
	예를 들어, a(b + c) = ab + ac와 같은 원리입니다.
	
	***
	
3. 결합 법칙(Associative Property with Scalar) :

	(cx)·y = c(x·y) = x·(cy)
	여기서 c는 스칼라(상수)입니다.
	도트 곱을 계산할 때 상수를 어느 벡터에 먼저 곱해도 결과가 같다는 의미입니다.
	
4. 크기 성질(Magnitude Property) :

	|x|² = x·x
	
	벡터의 크기의 제곱은 자기 자신과의 도트 곱과 같다는 성질입니다.
	
	이제 가장 중요한 부분입니다.
	왜 도트 곱의 두 가지 정의가 같은지 증명하는 과정을 살펴보겠습니다.
	첫 번째 정의 - x·y = x₁y₁ + x₂y₂ + ... + xₙyₙ
	두 번째 정의 - x·y = |x||y|cos(θ)
	
	증명 과정은 코사인 법칙을 사용합니다 -
	
	코사인 법칙 - c² = a² + b² - 2ab·cos(θ)
	벡터 차이를 이용 - z = y - x
	코사인 법칙을 벡터에 적용 - |y-x|² = |x|² + |y|² - 2|x||y|cos(θ)
	크기 성질과 분배 법칙을 적용하여 도트 곱으로 표현
	정리하면 결국 x·y = |x||y|cos(θ)가 됨을 보일 수 있습니다
	
	이 증명은 우리가 더 친숙한 첫 번째 정의를 기반으로, 기하학적 의미를 가진 두 번째 정의가 수학적으로 동등하다는 것을 보여줍니다.
```

	아, 아주 좋은 질문을 해주셨습니다! |x|² = x·x 는 단순한 제곱이 아닌, 매우 중요한 의미를 가지고 있습니다. 이를 단계적으로 살펴보겠습니다.
	먼저 2차원 벡터로 예를 들어보겠습니다.
	벡터 x = (a, b)가 있다고 해봅시다.
	
	벡터의 크기를 구하는 일반적인 방법:
	|x| = √(a² + b²)  [피타고라스 정리 사용]
	따라서 |x|² = a² + b²
	같은 벡터의 도트 곱을 계산해보면:
	x·x = (a,b)·(a,b) = aa + bb = a² + b²
	
	보이시나요? 두 결과가 같습니다!
	이것이 의미하는 바는 매우 중요합니다:
	
	벡터의 크기를 구할 때 피타고라스 정리를 사용하는 방법과
	벡터를 자기 자신과 도트 곱하는 방법이
	같은 결과를 준다는 것입니다.
	
	이는 도트 곱의 기하학적 의미를 더 깊이 이해하게 해줍니다. 벡터를 자기 자신과 도트 곱하면:
	
	각도는 0° (같은 방향)이므로 cos(0°) = 1
	따라서 x·x = |x||x|cos(0°) = |x|²
	
	이 성질은 나중에 더 복잡한 계산에서 매우 유용하게 사용됩니다. 특히 벡터의 크기를 계산할 때, 도트 곱을 이용하면 계산이 더 편리할 수 있습니다.
