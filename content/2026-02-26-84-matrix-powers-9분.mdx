# 84 Matrix Powers (9분)

부연설명 : 확률적 상태(마코프 행렬)에서의 예측

```yaml
응용 2: 확률적 상태(마코프 행렬)에서의 예측

이산 상태(Discrete states)와 확률 벡터

날씨 예측을 예로 들면, 하루의 날씨가 맑음(sunny), 비(rainy) 두 가지 상태 중 하나로만 정의된다고 가정
각 상태에 대한 확률을 벡터로 나타냄
예: X(t) = [ P(맑음일 확률), P(비가 올 확률) ]
전체 상태가 여러 가지(맑음, 흐림, 비, 눈, ...)라면 그 개수만큼 확률을 나열한 벡터가 됨
이 벡터의 각 원소들은 0 이상 1 이하의 값을 가지며, 모든 원소 합은 1
마코프 행렬(Markov Matrix) 개념

상태가 시점 t에서 t+1로 넘어갈 때, 해당 상태가 바뀔 전이 확률(transition probability)을 행렬 형태로 나타낸 것이 마코프 행렬
강의에서는 이 행렬을 a라고 두어,
X(t+1) = a X(t)
형태로 확률 벡터가 업데이트된다고 설명
예: 2가지 상태(맑음, 비)일 때 a가 2×2 행렬이라면,
a =
[ P(맑음→맑음), P(비→맑음) ]
[ P(맑음→비) , P(비→비) ]
(행렬의 구성은 모델 설정에 따라 행 또는 열의 합이 1이 되도록 정의)
확률 예측 과정

현재 시점 t의 상태 확률 X(t)을 알고 있다면,
X(t+1) = a X(t)
를 통해 바로 다음 시점(t+1)의 확률 벡터를 구할 수 있음
나아가,
X(t+2) = a X(t+1) = a² X(t),
X(t+p) = a^p X(t)
와 같이 p 스텝 뒤의 상태 확률을 한 번에 계산 가능
이때 a가 대각화 가능하다면(강의 전반부에서 설명한 diagonalization 활용) a^p 계산이 훨씬 단순화되어, 더욱 빠르게 예측을 수행할 수 있음
날씨 예측을 예로 든 실제 활용

“맑음”과 “비” 같은 이산 상태가 있을 때, 특정 기간 동안의 날씨 관측 데이터를 바탕으로 전이 확률(a)을 추정
이후에 오늘 날씨가 맑을 확률, 비 올 확률 등이 담긴 X(t)에서 시작해,
1일 뒤 X(t+1), 7일 뒤 X(t+7) 등의 미래 날씨 확률을 쉽게 예측
실제 기상청, 기후 모델링 등에서 이런 아이디어가 확장·응용되어 매일의 날씨나 강수 확률 등을 계산
다양한 산업 분야에서도 유사하게 활용

고객 이탈 예측(현재 서비스 이용 상태 → 유지·이탈 확률), 재무 분야(현재 시장 상태 → 상승·하락 확률), 유전학(유전자 변이 상태 확률) 등
일반적으로 다음 상태에 대한 불확실성을 수치화하고, 그 변화를 “확률적 전이 행렬”로 나타낼 수 있다면 모두 마코프 프로세스로 접근 가능
a^p 형태를 이용하면 여러 스텝 뒤의 확률 분포를 한 번에 산출할 수 있어 예측·분석에 효과적
정리

마코프 행렬 a를 이용해, 시점마다 상태 확률 벡터 X(t)을 업데이트
a^p를 통해 p 스텝 후의 확률을 직접 계산할 수 있어, 미래 상태 예측이 용이
날씨, 금융, 물리 시스템, 생물학 등 다양한 분야에서 활용됨
```
