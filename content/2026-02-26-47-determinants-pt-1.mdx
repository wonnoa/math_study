# 47 Determinants (pt 1)

```yaml
Determinant :

	- 2x2 matrix에서 ad - bc한 값
	
	- 여기서 ad는 “대각선 아래쪽 원소들의 곱” (down product)이고, 
	bc는 “대각선 위쪽 원소들의 곱” (up product)입니다. 이 두 곱의 차이가 행렬식의 값이 됩니다.

		주대각선(main diagonal): 왼쪽 위에서 오른쪽 아래로 가는 대각선 (a에서 d를 잇는 대각선)
		반대각선(counter diagonal): 오른쪽 위에서 왼쪽 아래로 가는 대각선 (b에서 c를 잇는 대각선)
		
특징 :

- scalar
- matrix에 대한 다양한 성질을 보여준다
```

```yaml
다양한 성질 1 :
	
	- matrix의 invertibility 판단 :
	
		A^1 = 1 / ad - bc  (-d b)
											 (-c a)
											 
		라는 식에서, ad - bc가 0이면, matrix는 singular 하다.
		
		ad - bc = 0의 의미는 ? :
			
			vector가 하나의 equation으로 겹침.
			차원의 축소.
			추가적인 차원 정보를 나타낼 수 없음.
			2차원에서 가로 1개 세로 1개가 나와야 하는데,
			가로든 세로든 정보하나가 손실된 상태.
```

```yaml
		X = A⁻¹B
```

```yaml
Linear System of Equations와 Slope의 관계 :

	1. 두 linear equation의 slope가 같으면 solution이 없거나 solution이 무한함 :

			없는 경우 - line이 평행
			무한한 경우 - 같은 line

			이럴 때, Det = 0
	
	2. 왜 Det = 0이면 Slope가 겹치는가 ?
	
		직선 1: Ax + By = C  →  y = (-A/B)x + (C/B)
		직선 2: Kx + Ly = D  →  y = (-K/L)x + (D/L)
		
		중요한 것은 각각의 slope 는
			
		(-A/B)
		(-K/L)
		
		Matrix A 는 

		 [A B]
     [K L]
     
     Det는
	    
	    det(A) = AL - BK
		
		det가 0이라면
	
			AL - BK = 0
			AL = BK
			
			A/B = K/L
			
			즉 slope는 같다
```
