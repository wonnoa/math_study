# 7 3 Equations and 3 Unknowns (17분)

- **One Solution :**

도대체 어떻게 세 평면이 `단 하나의 점에서만` 교차할 수 있는가?
이것을 시각적으로 상상하기는 어렵다.

하지만 단지 특정 규칙을 가진 숫자들의 모음을 공간위에 표시한 것이 평면이고,
이 세개의 equation들을 동시에 충족하는 단 하나의 집합을
공간위에 점으로 표시한 것이 그 교차점임을 인식하면된다.

그렇다면 핵심적인 질문은 다음과 같을 것 이다.

"어떤 경우에, "3개의 숫자들의 모음" 3개가 단 하나의 공통적인 해만 가질 수 있는가?"

교재에서 말하는,
one solution인 경우 점이고,
no solution인 경우 평행이고...
뭐 이런 것들은 그냥 그림을 그리면 설명되는 것들임.

정말 중요한 것은 각 경우에 대하여 숫자나 시각적 자료없이,
단순한 언어로만 설명할 수 있는 보편적인 원리를 이해하는 것임.


질문 :

"일반화시켜 설명할 수 있어야 한다"

'3개의 숫자 모음'이 3개 주어졌다. 이 숫자들간의 논리적 관계가 어떨 때

	- 해가 하나인가?
	- 해가 무한한가?
	- 해가 없는가?

그리고 그 이유에 대한 가장 본질에 가까운 좋은 설명은 무엇인가?

- 숫자 모음 :

[a   b   c  ] : k
[a'  b'  c' ] : k'
[a'' b'' c''] : k''

나의 이해 :

**
1. 해를 찾는다는 것의 의미 :
	
	뭔가 상호간 비교하여서 남는 차이를 대조하려고 노력하는 행위같다고나 할까.
	
	ax + by + cz
	dx + ey + fz
	gx + hy + iz

1. 해가 하나 : 세 모음의 원소들의 비율이 모두 다를 때.
2. 해가 무한 :
3. 해가 없음 : 


이것을 궁금해하는 이유 :

1. 무의식적인 행위를 언어로 기술하고 그것을 다시 이해하면 우리가 하는 행동을 다양한 상황에서 더 잘 응용할 수 있다.
2. 해가 몇 개인지의 상황이 왜 발생하는지 원인에 대해서 일반화하고 범용적으로 다룰 수 있다.

단 하나의 점에서만 이 Plane들과 교차하는 Plane 3을 상상한다.

의미적으로는 :

세 규칙을 모두 만족시킬 수 있는 set of "x,y,z"값을 찾는 것.

```yaml
그러니까, n개의 equation들이 한번에 만족되는 n개 변수들의 값을 각각 찾았다고하자.
그걸로 어쩌라고? 찾아서 뭐 어쩌라고?

관찰된 현상에서 주어진 시스템내의 변수들의 관계에 대한 보편적 진리를 찾는 행위 이다 :

"지능이 60인 아이가 공부를 4시간했더니 성적이 80점 나왔다",
"지능이 40인 아이가 공부를 6시간했더니 성적이 78점 나왔다",
"주어진 2개의 관계식에서, 보편적인 진리를 찾는 행위인건가?",
"n by n 관계에서 어쨌든 n개의 데이터는 있어야 시스템의 문제를 풀 수 있는 듯 하다",

	1. 우리가 찾고자 하는 것은, 주어진 시스템 규칙내에서 definite한 x와 y의 값.
		algebra에서 우리는 '모르는 값'을 x,y 등의 추상적인 기호로 표현해놓고 그들을 찾아간다.

	2. 하나의 equation만으로는 어떠한 경우도 가능함.
		- 무한한 직선이 그려질 수 있다.

	3. definite한 관계식을 주어진 시스템내의 주어진 데이터를 통하여 추론한다.

		예시 :

			4x + 5y = 75
			6x + 2y = 100

			공부를 4시간 했을 때, 지능이 5인 친구는 75점의 점수를 받더라.
			공부를 6시간 했을 때, 지능이 2인 친구는 100점의 점수를 받더라.

			공부시간과 지능의 성적에 대한 관계는 어떻게 되는가?
```
