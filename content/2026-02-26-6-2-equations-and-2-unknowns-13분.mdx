# 6 2 Equations and 2 Unknowns (13분)

  - 2 Equations, 2 Unknowns가 있을 때, unknown들을 찾아가는 두 가지 방법론.

```yaml
- Solving for x or y

	y = 2x - 1
	y = 1/2x + 2

	- 방법론 1. Substitution :

		x 나 y를 다른 변수로 표현하여 대입 하는 방법

	- 방법론 2. Elimination :

		하나의 함수에서 다른 함수를 빼서, 특정 변수의 영향력을 고립시켜
		논리적으로 해당 변수의 영향력을 추론하는 기법.

		한 방정식에 상수를 곱하여 같은 항과의 계수를 맞춤
		이후 같은 항목들을 eliminate함 (subtract함)

		예 :

		"2x + 2y = 2" 에서
		"2x - y  = 3" 를 뺀다

		그러면,
		2x는 서로 제거되고, 3y 남고, "="의 오른편에는 -1이 남는다.

		3y = -1이 된다.

		그럼

		y = -1/3이 된다

논리적 관점에서 System of Linear Equations를 푼다는 것은 :

		"두 개이상의 수식(변수들의 논리적 관계조건)을 '동시에' 충족시킬 수 있는 원소들의 집합을 찾는 것"

		Equation (수식)은 무엇인가 :

		"수식은 변수들의 상호작용 시스템"

		나는 수식을 변수들의 상호작용을 표현한 하나의 관계체계라고 이해한다.
		특정 변수가 어떠한 양적인 특징을 가질 때, 다른 변수가 결정적으로 가지게 될 양적인 특징을 가진다.
		그 논리적인 관계를 기술해놓은것이 수식이라고 생각한다.

		수식은 하나의 "규칙"이며 "시스템체계"이다.

		해가 존재할 때 'intersect'한다는 것의 의미 :

		"기하학적으로만 이해하는 것은 불온전하다고 생각한다, 기하학적인 설명은 부분적인 설명이다"

		해가 존재할 때 intersect한다 라는 설명은 순수한 논리적인 설명과 기하학적인 설명이 섞여있는
		지저분한 설명이라고 생각한다.

		각 equation을 만족시킬 수 있는 수의 집합들을 그래프라는 공간속에서 표현하였는데,
		이 때 equation들을 동시에 만족시킬 수 있는 수의 집합이 존재하므로
		이것이 점으로 표시된다는 설명이 조금 더 나은 듯 싶다.
		완벽한 설명을 하기 위하여서는 '공간'에 대해서 조금 더 탐구해야할 것.

one solution :

	두 선형 방정식이 동시에 만족되는 단 하나의 수의 집합이 존재.

no solution :

	두 선형 방정식이 동시에 만족되는 단 하나의 집합도 존재하지 않음.

	# 더 탐구할 지점 :
	solution이 없다는 것은 수들의 관계 측면에서 어떻게 설명되어야할까?

infinite solutions :

	두 선형 방정식을 동시에 만족하는 단 하나의 집합이 무한함.
	
2차원 선형에서 no solution 과 infinite solution은 시작점만 다를 뿐 본질적으로 같아보임 :

	no solution
		두 수식에서 x와 y의 변화율은 같고, 단지 시작점이 다름. (시작점 = 상수)

	infinite solution
		그냥 두 수식이 같은 수식임. 변화율 뿐 아니라, 시작점도 같다.

하나의 수식에 특정 scalar를 곱해도 같은 수식인 이유 :

	수식에서 우리의 관심사는 변수들간의 "비율 관계"이기 때문.

	수식을 scalar 곱하면 같은 비율이 여러개로 증가할 뿐,
	각 원소들 사이의 비율 자체는 바뀌지 않는다.

	- a,b
	- 2a,2b
	
	a:b 관계는 보존된다.

그래서 어쨌다고? :

	unknown을 찾아가는 방법에 대한 여러가지 설명이 존재한다.
	본질은 하나의 원소를 같은 수로 만들고,
	변화하는 다른 하나가 얼마나 변화하는지 그 차이를 보는 것.

	- 따라서 첫 번째 규칙을 만족하는 모든 점이 해가 됨
```

		만족하다의 의미 :

		"우리가 정한 수식이 정답이라고 성립되었다는 것"

		수식 1. ax + by + c = 0 에 대하여,
		특정 x,y 값을 넣었을 때, 0이 되는 조건.
		특정 x,y 값을 넣었을 때, 0이 되지 않는다면 해당 변수들은 이 식을 "만족"시킬 수 없다.


현실에서의 비유법 예시 "y = output" 관점 :
	"변화율의 거듭더하기 라는 본질적 태생에서 비롯되는 문제"

	현실에서 달리기를 한다고 치자.
		두 명의 속도 다른 사람들이 있음.
		그들의 속도는 변화율 (m), 출발점은 시작점 (c)

	둘이서 다른 시작점에서 출발하면, no solution. (절대 안 만남)
	둘이서 같은 시작점에서 출발하면, infinite solutions. (무한히 만남)

	우리가 변화율에 관심있느냐, 시작점에 관심있느냐의 차이일 것.
	보통 우리는 변화율에 관심이 있어서, "고정된 시작점"에 영향을 주는 것은 이 변화율.
	그런데 변화율이 같다는 것은, 절대로 같아질 수 없다는 운명. (y = output관점에서)
