# 58 Linear Independence and Dependence

```yaml
row가 상수배면 col도 상수배인 이유 :

사람이 한명, 팔이 두개.
사람이 두명, 팔이 네개. 의 의미는

사람이 한명, 팔이 두개
사람이 한명, 팔이 두개
사람이 한명, 팔이 두개

사람 하나당 팔이 두개라는 관계성 유지되고 있음.
row가 상수배라는 것의 의미는, 데이터간의 관계식이 유지되고 있다는 뜻.
그것이 복제된다면, 그 관계성은 col간에서도 유지됨.

Linear Independence :

"linearly independent한 vector들로, 해당 rank의 차원의 span을 표현할 수 있다",
"linearly indpendent한 vector는 표현되지 못했던 새로운 vector 값을 도출해낼 수 있다",
"꼭 하나의 row나 column이 다른 것의 상수배여야지만 linearly dependent한 것은 아니다",
"linearly independent 는 꼭 vector들이 orthogonal하다는 의미는 아니다"

조심 :

	"2x2 matrix 에서 linear dependence 성립은 언제나 하나의 vector가 다른 것의 상수배",
	"3x3 matrix 이상에서는 linear dependence 성립은 항상 상수배일 필요는 없고, 하나가 다른 것들의 linear combination"
		[1 0 0]
		[0 1 0]
		[1 1 0]
		-> 아마 2x2는 선형 종속 숫자를 구할 수 있는 방법이 상수배인 경우밖에 없어서 제한되는 것 같고,
		3x3이상부터는 방법이 다양해서 그런 듯.
		linear independence의 정의 자체가 완전히 새로운 정보를 만들어낼 수 있느냐 이니까.
		
예 :
	(2,1)은 linearly independent 한가?
	
	- (1,1)은 (1,0)과 (0,1)의 linear combination (scalar multiplication and addition of vectors)
	으로 표현할 수 있다.

```

```yaml
Vector 관점 :
	
	Linear Combination :
		
		vector에 scalar를 곱하고 더한 결과.
		(1,0),(4,2) => 4(1,0) + (4,2)
		
	0 vector :
		
		모든 원소가 0인 vector
			[0,0,0,...]
	
	Linear Combination이 0 vector 의 의미 :
		
		vector set에 scalar를 곱하고 더한 결과가, 0 vector.
		
			
	Trivial Solution :
	
		linear combination a₁·v₁ + a₂·v₂ + … + aₙ·vₙ = 0벡터가
		모든 계수 a1,a2... = 0일 때에만 성립하는 경우.
	
		trivial solution의 의미 :
			 
			1. 0벡터라는 것은, 수식에서 벡터들의 정보가 모두 상쇄되었다는 뜻.
			2. 특정 계수를 하나의 벡터에 곱하여 다른 벡터정보를 지울 수 있다면, 
				이는 벡터가 서로 상수배임을 의미.
			3. 특정 벡터들끼리의 결합을 통해 다른 벡터의 정보를 지울 수 있다면,
				이는 벡터간의 linear combination으로 다른 벡터를 표현할 수 있음을 의미.
		
	선형 독립의 조건 :
	
		어떤 벡터 집합이 선형 독립이라고 말하는 것은, 
		위의 선형 결합이 0벡터가 되는 유일한 방법이 계수들이 모두 0일 때뿐임을 의미
	  만약 그 외에 다른(0이 아닌) 계수를 사용해서도 0벡터를 만들 수 있다면, 
	  그것은 벡터들 사이에 중복(종속)이 있다는 뜻.
	  
	  **햇갈리지말것 : 꼭 두 벡터가 90이어야 선형독립이 아니다.
```

```yaml
Matrix 관점 :

	Linear Combination :
	
	"Matrix 관점에서 linearly independent 하다는 것은, 행렬의 행(또는 열) 벡터들이 서로 독립적인 정보를 제공한다는 것을 의미합니다."
		
	Matrix관점은 Matrix의 각 행과 열을 독립적인 벡터에서 보는 관점이다 :
	
		Matrix의 각 행,열을 독립적인 벡터로 생각.	
	
		서로에게 상수배가 아닌 equation들은 독립된 정보를 제공함.
		따라서 linearly independent함.
		
	Matrix에서 특정 column이 다른 column의 상수배라는 것은 :
		
		하나의 vector 정보가, 다른 vector 정보에 상수배, 즉 종속적이라는 뜻.
		 * 데이터 사이에서 새로운 관계에 대한 정보를 주지 않는다.
	
			**Vector Set A** =
				
				v1 v2
				[1,3]
				[2,6]
			
			->> v2의 x,y관계는 정확히 v1의 x,y관계와 동일하여 추가적인 정보를 제공할 수 없다.
		
		신기한것은 column이 상수배이면, row도 상수배가 된다는 점.

		row :
		
		"row를 equation관점으로 볼 수 있다"
		
		Matrix에서 하나의 row는 하나의 equation과 같음.
		하나의 row가 다른 row의 n배이면, 새로운 관계 규칙에 대한 정보를 제공하지 않는다.
				
		column :
		
		"column을 하나의 독립적인 정보set으로 볼 수 있다"
				
		Matrix에서 하나의 column은 하나의 정보 set과 같다.
		하나의 column이 다른 column의 n배이면, 실상은 같은 정보이다.
			(새로운 데이터가 없다 - 어린 원노아나 다 자란 원노아나 유전자는 같다)

		그것들을 정보이던 Equation으로 보던간에 상관없이,
		그들은 모두 x,y,z... 등의 변수들간의 "관계"를 밝혀내는 데이터이다.
	
	그렇다고 한다 :
	
	"일반적으로 선형 독립성을 판단할 때는 방정식의 계수(좌변)만 고려합니다."
```

```yaml
Linearly Independent :
	
	"Matrix 관점"
	
	Matrix A의 row와 column이 linearly dependent이어야 전체 공간을 표현할 수 있다.
	또한 그래야만 det가 0이 아니게 된다.
	
	no solution과 infinite solution의 문제 :
	
		equation으로 표현하고자하는 n차원의 공간을 모두 표현할 수 없게됨.
		
	Det 와의 관계 :
		
		"det가 0이면, linearly dependent하다고 한다"
		
		A = [2,1;4,2]의 경우 두 번째 행은 첫 번째 행의 2배.
		Det A = (2x2) - (4x1) = 0 이므로, invertible하지 않음.
		
		이는 변환을 통해, 하나의 vector가 다른 vector에 종속되어버림을 의미.
```

```yaml
Linear Combination 과 Linearly Independent :

	"Vector 관점"
	
	하나의 벡터를 상수배하여도 다른 벡터가 될 수 없다.
	
		이는 기하학적으로 아래를 의미한다 :
		
		"주어진 공간을, 주어진 벡터로 채울 수 있어야 한다. 잉여 벡터는 필요없다"
		
		2차원: 면적이 0이 아닌 평행사변형을 만들 수 있어야 함

	Linear Combination으로 다른 Vector를 표현할 수 있다면,
	해당 vector는 linearly dependent하다.

	차원관점에서 특별하게 어떤 새로운 정보를 주는 정보는 아니라는 것.
```

```yaml
official term of linear independence :

	벡터 집합 {v₁, v₂, ..., vₘ}이 있을 때,
		α₁v₁ + α₂v₂ + ... + αₘvₘ = 0
	
		이 방정식에서:
	
		모든 α = 0일 때만 등식이 성립 → Linear Independent
		0이 아닌 α값들로도 등식 성립 → Linear Dependent
	
	이것이 의미하는 것:
	
		Independent case:
	
		어떤 벡터도 다른 벡터들의 조합으로 표현 불가
		각 벡터가 고유한 방향/정보를 가짐
		전체 공간을 효과적으로 표현 가능
	
		Dependent case:
	
		최소 하나의 벡터가 다른 벡터들의 조합으로 표현 가능
		v₁ = β₂v₂ + β₃v₃ + ... + βₘvₘ 형태로 표현 가능
		이는 중복된 정보를 의미
	
		대수적 해석:
	
		만약 v₁이 다른 벡터들의 조합이라면:
		
		v₁ = β₂v₂ + β₃v₃ + ... + βₘvₘ
		↓
		v₁ - β₂v₂ - β₃v₃ - ... - βₘvₘ = 0
		↓
		이는 α₁ = 1, α₂ = -β₂, α₃ = -β₃, ..., αₘ = -βₘ
		인 경우가 존재함을 의미
		
		
```

