# 8 Gaussian Elimination (23분)

  **Gaussian Elimination**

```yaml
  주요 규칙

  	Rule 1: row에 상수 곱하기 가능
      
      소거라는 과정의 특징때문에 가능하다.
      
      [a,b,c] [x]
      [d,e,f] [y]
              [z]

      소거는 행의 각 위치의 숫자 원소들이 같은 값의 x,y,z와 곱해질 것이라고 가정함.
      x,y,z가 어떤 숫자와 곱해지는지에 따라 달라지는 값의 차이를 측정하여 각 변수의 유일한 값을 측정하려고 시도함.
      이 때, 소거는 각 행에서 하나의 원소의 값을 같게 만들어서 해당 원소에 곱해지는 변수의 영향력을 제거한 뒤,
      나머지 변수들의 값을 찾으려 시도함.

      따라서 각 행의 원소들의 비율적 특성이 보존되기만하면 (상수배),
      다시 상쇄하려는 변수를 같게 만들 것 이므로 소거하여 원하는 값을 찾아나가는 것이 가능해짐.
      
  	Rule 2: 한 row를 다른 row에 빼는 것 뿐 아니라 더하는 것도 가능
      
      중요한 것은 같은 변수의 영향력을 각각의 행에서 상쇄시켜서 측정 결과에서 없애는 것임.
      변수의 영향을 상쇄시킬 수 있으면 -2에 2를 더하여 0으로 만들어도 전혀 무방함.

  	Rule 3: row 위치 교환 가능
      
      당연함, 같은 변수에 대하여 다루므로 어떤 수식을 먼저 처리하든지 상관없다.
  ```

  이것이 원래의 형태와 같은 것인가?

  수식 그 자체로는 엄연히 다르다.
  하지만 이 상황에서 주어진 변수값들의 영향력을 같다.

  다만, 그것이 수학, 즉 "수를 센다"라는 것, 그리고 선형 수식 관점에서 어떤 의미인지 정말 어렵군요

  ```yaml

  Gaussian Elimination에서 하나의 변수를 "소거"하고자 하는 이유 :

  1. 각 방정식은 하나의 독립적인 제약조건 :

  x + y + z = 2
  6x - 4y + 5z = 31

  수학적으로, 각 방정식의 계수는 "해당 변수가 1 변할 때 전체 식에 미치는 영향"을 나타냅니다. 

  첫 번째 식에서 x가 1 변하면 결과도 1 변합니다. 
  두 번째 식에서 x가 1 변하면 결과는 6 변합니다.
  소거법의 핵심은 "한 변수가 각 방정식에 미치는 영향을 동일하게 만든 뒤, 그 영향을 제거"하는 것입니다.

  예를 들어, 첫 번째 식에 6을 곱하면,
  6(x + y + z) = 6(2)
  6x + 6y + 6z = 12

  이제 두 식에서 x는 동일한 영향(6)을 미칩니다. 이 상태에서 두 식을 빼면,
  (6x + 6y + 6z) - (6x - 4y + 5z) = 12 - 31

  1-1. 특정 변수의 영향을 "상쇄"시켜도 되는 이유 :

  어차피 우리는 두 수식 모두를 만족하는 해의 집합을 찾고 있다.
  따라서, 두 수식에서의 x 값들은 같을 때 y,z값은 무엇이 될지를 전재로하고 있음.
  따라서, 6x를 지워버리면, 나머지 y와 z만 탐구할 수 있다.
  ```

  "만약 첫 번째 규칙이 성립하고, 두 번째 규칙도 성립한다면, 그 차이도 반드시 성립해야 한다"

  부가 설명 :

  1. 수식 2개가 있다고 친다.

  	x + y - 5 = 0
  	2x + 3y - 19 = 0

  2. 각 Equation은 "시스템", "규칙 체계" 이다.

  	우리가 찾고자 하는 것은,
  	두 시스템이 모두 성립할 수 있는 x,y의 set을 찾는 것.

  3. "연립방정식"의 의미는 바로, 두 규칙을 가진 새로운 system을 창조한다는 것 이다.

  	그렇지.. 이것을 이해해야, 비로소 이것을 "머신러닝"이든 어떤 분야에든 확장시켜서 사용할 수 있다고 믿는다.

  4. 

  아직 남아있는 내 궁금증은, 어째서 Equation들을 등식에 "합치고, 빼면서" 새로운 정보를 도출해도 되느냐.

  중요한 것은, 이것이 도대체 어떤 의미인가... 라는 것이지...

  x,y를 찾는 작업을 진행중이라는 것이고, 그 과정에서 하나의 equation을 통해서, 다른 equation의 정보를
  도출해내겠다 라는 작업이다.

  하나의 등식에서의 x와y값이 다른 등식에서 같을 것임을 가정하였기 때문에,
  해당 Equation들을 합성하여 정보를 얻어내도 되는 것이지 않나 싶다.
  "합성"한다는 것은, 대입, 소거 등의 작업을 말한다.
  마치 주어진 Equation들이 같은 것처럼 대하는 것.

  그냥, 숫자만 등호 사이로 왔다갔다 거리면서 맞죠 맞죠, 이렇게 증명해도 되긴한다..
  그런데, 글쎄.. 그것이 정말 "수"를 센다는 것의 의미를 반영하는 행위는 아닌 것 같고,
  그냥 공허한 규칙놀음?이라는 생각이 강하게 든다는 것..
  물론 규칙놀음이 의미 이전의 더 본질적인 것일 수 있다. 규칙은 절대적이니까.
  하지만, 우리에게는 이것들에게 의미를 부여할 수 있어야 진정으로 가치가 있다고 생각한다.

  그냥 머신러닝에 필요한 수학의 계산법만 배워서, 이후에 실무에서 활용하고 있으면,
  그냥 내가 하고 있는 계산이 어떤 수학적의미를 가지는지도 모른 채, 의미없는 숫자놀음만
  뚜닥뚜닥하면서 와 공식이 성립했네만 반복하고 있을 것 같았다.
  그래서 최대한 수들의 본질을 이해해보려고 노력하고 있다.
  수학적 규칙을 위배하지 않는 어떠한 계산 결과든지 합성될 수 있지만, 그것이 현실에서 진정한 의미를 가진다고
  생각하지 않기 때문.

  (고등학교때도 그냥 계산 당연히 할 수 있는데, 이런 생각하느라 시간 다보냈고,

  2x + 2b - 10 = 0 이라는 식에서 우변의 0은
  특정 x와 b를 가졌을 때 0이 되면 이 식에 대한 규칙이 만족되었다 라는 마킹포인트로 이해해도 된다.

  2x + 3b - 19 = 0 이라는 식도 마찬가지.

  x의 영향력은 이제 배제하였으니, 
  두 수식에서 b가 모두 같은 값이라는 전제할 수 있는 상황속에서,

  	2b - 10 = 0
  	3b - 19 = 0

  2b 이면 - 10을 해야 수식이 만족되는 상황이고,
  3b 이면 - 19를 해야 수식이 만족되는 상황이다.

  그럼 논리적으로 유추하건데, b하나의 영향력 차이는 여기서 9라고 할 수 있다.

  최종 설명 :

  system A에서 2x + y 는 정확히 14가 되어야 한다.
  system B에서 x - y 는 정확히 2이다.

  우리는 두 가지 규칙을 모두 만족할 수 있는 x와 y를 찾으려 하는 중이다.
  왜냐면 두 개의 규칙이 "합성"되어 새로운 시스템체계를 생성할 수 있기 때문이다.

  어떻게 소거법의 원리를 이해할 수 있을까?를 생각해보았다. :

  소거법과 대입법을 애초에 배우지 않았다고 가정해보았다.
  그냥 단순히 이 규칙들만 주어졌을 때, 어떻게 문제를 논리적으로 풀어나갈지 고민해보았다.
  그랬더니 소거법을 이해할 수 있었다.
  어차피 소거법 모르는 사람들이 같은 문제를 해결하려고 만들어낸 것이 소거법 아니겠는가?

  수식으로 하면 :

  2x + y = 14
  x - y = 2

  Elimination은 왜 해도 되는걸까? :

  0) 수학에서의 equation은 각 수들이 각각에게 미치는 영향력에 대한 규칙임을 상기한다.
  1) 두 규칙속의 x와 y가 같다고 전제하고 있다는 것을 상기한다.
  2) 같기 때문에, 변수 하나하나를 같게 만들어 그들의 영향력을 제거,
  	 최종 하나의 변수의 영향력을 논리적으로 추론 및 측정할 수 있기 때문이다.

  뭐라는거야 :

  두 규칙이 같은 x와 y를 가져야 한다고 전제하고 있기 때문에 가능하다.
  전제함으로, 위 규칙과 아래 규칙의 변수들의 수가 "같다고" 생각해버리는 것이다.
  만약 같다고 전제하지 않으면, 두 규칙에서의 수들은 그냥 다른 수들일 뿐이다.
  그리고 우리의 "전제하고 있는 상태"는 수식에 보이지 않는다. (우리 생각속에 있어서 안보임)

  실상 위 규칙에서 x와 y는 다른것일 수 있지만, 지금 이상황에서는 같은것으로 가정한다.

  그럼, x는 상상속에 정해진 하나의 수 이니까, 아래 수식에 2를 곱하여서 수식을 비교하기 쉽게 만들어보자.

  2x + y = 14
  2x - 2y = 4

  그럼 이 2x가 무엇인지는 알 수 없지만, 어쨌든 그들의 영향력은 위 아래 두 규칙에서 모두 같다.
  (왜냐면 우리가 같다고 전제하고 있으니까)

  그럼 이제, 그냥 그들을 "아무 수"라고 생각하고 각각의 규칙에서 지워버릴 수 있다.
  y의 영향력만 측정할 수 있는 상황이 온다.

  해보면 :

  아무수 + y = 14
  아무수 - 2y = 4

  아무수에 y를 더하면 14라는 수가 나오고, 아무수에서 2y를 빼면 4가 남는다.
  그럼 이제 논리적 추론이 가능해진다.

  y한번가면 14가 되고, -2y가 줄어들면 4가 된다.

  y를 추론해보면 :

  1) "아무수"는 4와 14의 중간일 것이다. (y를 한번 더하면 최대 14, 2번빼면 4니까 중간이겠지)
  2) 중간값은 14와 4의 차이에서 3으로 나눈 뒤 y 비율만큼 분배하면 된다.
  	 4부터 14의 차이는 10이라는 차이가 존재한다. 그런데 이것을 3등분하면 10/3이 됨.
  	 우리가 움직일 수 있는 y는 3칸이 존재, 그리고 그 범위는 10이라는 숫자.
  	 그럼 10을 3으로 나눔. 4에서 10/3을 두 칸 위로가면 "아무수", 14에서 10/3을 한칸 내려오면 "아무수".
